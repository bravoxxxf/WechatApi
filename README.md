#### · 使用最流行的Mysql数据库
#### · 热更新（src/api）
#### · 支持小程序授权（xcx_autho.js）
#### · 更多功能待更新...

# 使用方法

## 下载脚手架
```
$ git clone git@github.com:xiaofei1990/WechatApi.git
```

## 安装依赖
```
$ cd WechatApi
$ npm install
```

## 开始运行
```
$ npm start
```

## 停止运行
```
$ npm stop
```
